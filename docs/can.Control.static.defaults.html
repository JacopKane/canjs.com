defaults

can.Control.static.defaults

A object of name-value pairs that act as default values for a control's this.options.

Message = can.Control({
  defaults: {
    message: "Hello World"
  }
}, {
  init: function(){
    this.element.text( this.options.message );
  }
});

new Message( "#el1" ); //writes "Hello World"
new Message( "#el12", { message: "hi" } ); //writes hi

In can.Control.prototype.setup the options passed to the control are merged with defaults. This is not a deep merge.
