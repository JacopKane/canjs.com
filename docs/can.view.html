can.view

can.view(view, data, helpers)

Parameters

  1. view {String}

    The URL to a template or the ID of a template.

  2. data {Object}

    Data to populate the template with.

  3. helpers {Object}

    Helpers referenced in the template.

Returns

{String}

The template with the data interpolated into it.
