can.remove

Remove elements from the DOM.

can.remode(nodeList)

Parameters

  1. nodeList {NodeList}

    A nodelist of elements to remove.

can.remove( wrappedNodeList ) removes the set of matched element(s) from the DOM.


can.remove(can.$('#wrap')) //-> removes 'wrap'
