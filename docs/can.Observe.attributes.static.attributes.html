attributes (static)

 

can.Observe.attributes is a property that contains key/value pair(s) of an attribute's name and its respective type for using in and .

var Contact = can.Observe({
    attributes : {
        birthday : 'date',
        age: 'number',
        name: 'string'
    }
});
