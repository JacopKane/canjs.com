jQuery.fn.control

Get the Control associated with elements.

jQuery.fn.control([type])

Parameters

  1. control {String | can.Control}

    The type of Control to find.

Returns

{can.Control}

The first control found.

This is ths same as $().controls except that it only returns the first Control found.

//- Init MyBox widget
$('.widgets').my_box();

<div class="widgets my_box" />

$('.widgets').controls() //-> MyBox
