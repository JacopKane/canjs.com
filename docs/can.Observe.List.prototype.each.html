each

can.Observe.List.prototype.each

Call a function on each element of a List.

each(callback)

each iterates through the Observe, calling a function for each element.

Parameters

  1. callback {function}

    the function to call for each element The value and index of each element will be passed as the first and second arguments, respectively, to the callback. If the callback returns false, the loop will stop.

Returns

{can.Observe.List}

this List, for chaining

var i = 0;
new can.Observe([1, 10, 100]).each(function(element, index) {
    i += element;
});

i; // 111

i = 0;
new can.Observe([1, 10, 100]).each(function(element, index) {
    i += element;
    if(index >= 1) {
        return false;
    }
});

i; // 11
